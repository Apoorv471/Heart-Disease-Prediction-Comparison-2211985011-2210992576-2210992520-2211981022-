## 📖 Project Overview

This research project focuses on the early prediction of heart disease using Machine Learning techniques. Multiple classification algorithms were implemented and compared to analyze their performance in disease prediction.

The project also evaluates the impact of feature engineering approaches including:

* Chi-Square Feature Selection
* Recursive Feature Elimination (RFE)
* Principal Component Analysis (PCA)

The study aims to identify the most effective model for accurate and reliable heart disease prediction.

---

## 🎯 Objectives

* Predict heart disease using Machine Learning.
* Compare multiple classification models.
* Improve prediction performance using feature selection.
* Analyze dimensionality reduction techniques.
* Reduce false negatives in medical diagnosis.
